Perguntas sobre Desenvolvimento Frontend

1. O que é a compactação de recursos (resource minification) no contexto do desenvolvimento frontend?

a) Uma técnica para reduzir o número de requisições HTTP.

b) Uma técnica para tornar os recursos (arquivos) menores, removendo espaços em branco e comentários.

c) Uma técnica para melhorar a acessibilidade do site.

d) Uma técnica para melhorar a compatibilidade do site com navegadores antigos.

2. Qual das seguintes práticas ajuda a melhorar o tempo de carregamento de uma página web?

a) Uso extensivo de imagens em alta resolução.

b) Carregar todos os scripts JavaScript no cabeçalho da página.

c) Utilizar uma CDN (Content Delivery Network) para servir arquivos estáticos.

d) Incluir vídeos incorporados diretamente na página.

3. O que é uma técnica de "lazy loading" em relação ao carregamento de imagens?

a) Carregar todas as imagens imediatamente quando a página é aberta.

b) Carregar as imagens apenas quando o usuário clica em um botão específico.

c) Carregar apenas o conteúdo que está visível na área de visualização do usuário.

d) Carregar as imagens em um loop contínuo para uma animação suave.

4. Qual é a importância de fornecer alternativas textuais (text alternatives) para elementos não textuais em uma página web?

a) Melhorar o desempenho do site.

b) Tornar o site mais atraente visualmente.

c) Melhorar a acessibilidade para usuários com deficiência visual.

d) Reduzir a quantidade de código HTML na página.

5. Qual dos seguintes recursos do HTML5 é usado para tornar uma página web mais acessível para dispositivos móveis?

a) <meta name="viewport">

b) <nav>

c) <footer>

d) <section>

6. O que significa a sigla "SEO" no contexto do desenvolvimento web?

a) Search Engine Optimization (Otimização para Mecanismos de Busca)

b) Secure and Encrypted Operations (Operações Seguras e Criptografadas)

c) Server Error Options (Opções de Erro do Servidor)

d) Systematic Email Organization (Organização Sistemática de E-mails)

7. Qual das seguintes declarações é verdadeira sobre o uso de cache em um site?

a) O cache deve ser evitado, pois torna o site mais lento.

b) O cache é útil apenas para dispositivos móveis.

c) O cache ajuda a reduzir o número de requisições ao servidor e melhora o tempo de carregamento.

d) O cache não afeta o desempenho do site.

8. Qual é a vantagem de usar o pré-processador Sass (Syntactically Awesome Stylesheets) em vez do CSS tradicional?

a) Sass é mais fácil de aprender para iniciantes.

b) Sass oferece recursos adicionais como variáveis, aninhamento e mixins.

c) CSS tradicional não é suportado pelos navegadores modernos.

d) Sass não requer uma etapa de compilação antes de ser usado no navegador.

9. O que é o conceito de "above the fold" em relação ao design de páginas web?

a) É a área visível da página sem a necessidade de rolagem.

b) É um termo relacionado à acessibilidade de dispositivos com tela dobrável.

c) Significa posicionar os elementos de design apenas acima do rodapé da página.

d) Refere-se ao conteúdo que fica abaixo do rodapé da página.

10. Qual é a importância de um design responsivo em desenvolvimento frontend?

a) Melhora a classificação do site nos mecanismos de busca.

b) Garante que o site tenha acesso a todas as funcionalidades do dispositivo do usuário.

c) Mantém a consistência visual do site em diferentes tamanhos de tela.

d) Reduz o tempo de carregamento da página.

11. O que são os "web components" (componentes web) no desenvolvimento frontend?

a) Uma biblioteca popular de animações em JavaScript.

b) Uma técnica de otimização de imagens para a web.

c) Um conjunto de elementos HTML personalizados com funcionalidades reutilizáveis.

d) Um método para compactar vários arquivos CSS em um único arquivo.

12. Por que o uso de uma fonte web é preferido em relação a uma fonte instalada no computador do usuário?

a) Fontes web são sempre gratuitas, enquanto fontes instaladas podem exigir licenças.

b) Fontes web têm melhor desempenho de renderização em todos os navegadores.

c) Fontes web garantem uma maior consistência de design em diferentes dispositivos.

d) Fontes instaladas podem causar problemas de acessibilidade em alguns sistemas operacionais.

13. Qual é o objetivo do protocolo HTTPS em um site?

a) Aumentar a velocidade de carregamento da página.

b) Proteger a conexão entre o navegador do usuário e o servidor, garantindo a segurança dos dados.

c) Melhorar o desempenho do site em dispositivos móveis.

d) Fornecer suporte para recursos offline.

14. Quais são os atributos "aria-" em HTML usados para?

a) Definir o layout e estilo de uma página web.

b) Adicionar informações adicionais para dispositivos móveis.

c) Melhorar a acessibilidade para tecnologias assistivas, como leitores de tela.

d) Incluir ícones e imagens em uma página web.

15. Qual das seguintes opções NÃO é uma prática recomendada para melhorar o desempenho de um site?

a) Usar imagens em alta resolução para melhorar a qualidade visual.

b) Minimizar o uso de recursos externos de terceiros.

c) Comprimir e otimizar imagens antes de carregá-las no site.

d) Utilizar a técnica de "lazy loading" para carregar conteúdo à medida que o usuário rola a página.
